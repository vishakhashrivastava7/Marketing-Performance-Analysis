📊 Marketing Performance Analysis Dashboard

An interactive marketing analytics dashboard built using Power BI, Python, SQL, and Excel to analyze campaign performance and business insights.

🚀 Project Overview

This project focuses on analyzing marketing campaign data to track key performance metrics such as:

ROI (Return on Investment)
CTR (Click Through Rate)
CPC (Cost Per Click)
Impressions
Clicks
Conversions
Channel-wise Performance

The dashboard provides interactive visualizations and insights to help understand campaign effectiveness across different marketing channels.

SQL

📌 Dashboard Features
KPI Cards for business metrics
Funnel Analysis
Monthly ROI & CTR Trend Analysis
Channel-wise Click Analysis
Donut Chart for Channel Contribution
Interactive Date Slicer
Performance Summary Table
📈 Key Insights
Generated over 1 Billion impressions across campaigns.
Maintained an average CTR of ~14%.
Funnel analysis highlighted conversion drop-offs.
Email and Website channels showed strong engagement.
Interactive filtering improved campaign performance analysis.
